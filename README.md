ng-quick-table
=====================
### 功能:
 - 快速过滤记录
 - 平面太窄的时候，将3列合并成一列
 - 排序记录
 - 自定义表格元素模板
 - 自定义表格header模板
 - 双击编辑
 - 显示合并列
 - 导出CSV文件（Excel）
 - 更多功能和文档即将完成！！
